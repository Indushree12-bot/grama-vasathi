# Grama Vasathi (Hospitality)

## Overview

Grama Vasathi is an Android-based hospitality and accommodation booking application developed using Android Studio. The application helps users search, view, and book accommodation services through a simple and user-friendly interface. The project also integrates Firebase for backend support and Generative AI features for intelligent assistance.

This project was developed as part of Android application development learning and project implementation.

---

# Features

* User Registration and Login
* Search and View Accommodation Details
* Room / Hotel Booking System
* Firebase Authentication
* Real-Time Database Integration
* AI-Based Assistance and Recommendations
* Simple and Modern User Interface
* Android Mobile Support

# System Modules

## 1. User Authentication Module

This module allows users to register and log in securely using Firebase Authentication.

## 2. Accommodation Search Module

Users can browse available accommodations and search based on their requirements.

## 3. Booking Module

This module handles booking functionality and stores booking details in Firebase.

## 4. AI Assistance Module

Generative AI is used to provide intelligent suggestions and improve user interaction.

## 5. Database Module

Firebase Realtime Database / Firestore is used to manage user and booking data.

# Installation Steps

## Prerequisites

Before running the project, make sure you have:

* Android Studio installed
* Firebase account
* Android SDK installed
* Internet connection

---

## Clone the Repository

```bash
git clone https://github.com/your-username/your-repository-name.git
```

---

## Open in Android Studio

1. Open Android Studio
2. Click on **Open Project**
3. Select the cloned project folder
4. Allow Gradle to sync completely

---

## Firebase Setup

1. Create a Firebase project
2. Connect the Android app with Firebase
3. Download the `google-services.json` file
4. Place the file inside:

```text
app/google-services.json
```

5. Sync the project again

---

# Running the Application

1. Connect an Android device or start an emulator
2. Click on **Run ▶** in Android Studio
3. The application will build and launch automatically
